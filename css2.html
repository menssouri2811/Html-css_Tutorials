Propiedades

  1. color

    La propiedad color sirve para establecer un color para el texto. Los valores que puede llevar son:

    • Valores hexadecimales (#fcff50 un color verde).
    • Valores en rgb :rgb(252, 255, 82) es el mismo color anterior.
    • Tambien hay una gran lista de nombres de colores que se usan como valores (red,green,aquamarine...).

    Sintaxis
    p{color:#29d80a;}

    Color en hexadecimale.

    p{color:rgb(41, 214, 10);}

    Color en rgb.

    p{color:blue;}

    Color con nombre como valor.

  2. background-color

    Tiene la misma funcion que color cambiando el color del fondo el lugar del texto y lleva tambien los mismos valors.

    Sintaxis
    p{background-color:#29d80a;}

    Color en hexadecimale.

    p{background-color:rgb(41, 214, 10);}

    Color en rgb.

    p{background-color:blue;}

    Color con nombre como valor.

  3. width y height

    Las propiedades height y width se utilizan para establecer la altura y el ancho de un elemento. Las propiedades de alto y ancho no incluyen relleno, bordes ni márgenes. Establece la altura / ancho del área dentro del relleno, borde y margen del elemento. Se las puede asignar valores en %,px,cm,em....

    Sintaxis
    p{width:25px;height:30px;}
    p{width:25%;height:30px;}

  4. Border

    Las propiedades border de CSS le permiten especificar el estilo, el ancho y el color del borde de un elemento.

    • Propiedades de border

      • border-style

        La propiedadborder-style especifica qué tipo de borde a mostrar. Se permiten los siguientes valores:

        • dotted - Define un borde punteado
        • dashed - Define un borde punteado
        • solid - Define un borde sólido
        • double - Define un borde doble
        • groove - Define un borde acanalado en 3D. El efecto depende del valor del color del borde.
        • inset - Define un borde insertado en 3D. El efecto depende del valor del color del borde.
        • none - No define ningún borde
        • hidden - Define un borde oculto
        Sintaxis
        p{border-style:dotted;}

        border dotted

        p{border-style:dashed;}

        border dashed

        p{border-style:solid;}

        border solid

        p{border-style:double;}

        border double

        p{border-style:groove;}

        border groove

        p{border-style:inset;}

        border inset

      • border-width

        La propiedad border-width especifica el ancho de los cuatro bordes. El ancho se puede establecer como un tamaño específico (en px, pt, cm, em, etc.) o usando uno de los tres valores predefinidos: delgado, medio o grueso:

        Sintaxis
        p{border-width:10px;}

        border con 10px de width

        p{border-width:5px;}

        border con 5px de width

      • border-color

        La propiedad border-color se utiliza para establecer el color de los cuatro bordes.

        Sintaxis
        p{border-color:red;}

        border red

        p{border-color:black;}

        border black

      • border-radius

        La propiedad border-radius se usa para agregar bordes redondeados a un elemento

        Sintaxis
        p{border-radius:10px;}

        border con 10px de radius

        p{border-radius:5px;}

        border con 5px de radius
